# Elevate Visuals Media Website

## How to put it online on Vercel

1. Download this zip file and unzip it.
2. Go to Vercel and sign in.
3. Click **Add New Project**.
4. Drag the unzipped folder into Vercel.
5. Click **Deploy**.

## Files included
- `app/page.jsx`
- `app/layout.jsx`
- `app/globals.css`
- `package.json`
